# ex21 a-plus-twice-b

함수 `a_plus_twice_b()` 를 완성하시오. `b`에 2를 곱한 값과 `a`의 합을 반환하도록 하시오.<br>
Complete the function `a_plus_twice_b()`.  Make it return the sum of `a` and the twice of `b`.
